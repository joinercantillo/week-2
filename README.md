This is a Python script designed to manage a basic product inventory through an interactive menu. Here is a concise summary of its functionality:
Project Overview

Data Structure: It uses a list to store multiple dictionaries, where each dictionary represents a product with its name, cost, and quantity.

Looping Logic: A while loop keeps the program running until the user selects option 4 (EXIT).
Key Features

Product Entry (Option 1): Allows the user to input a product's name, price, and stock. It includes a nested loop so you can add multiple items without returning to the main menu.

Inventory Display (Option 2): Iterates through the list and prints each stored product. It includes a check to see if the list is empty.

Statistics (Option 3): Calculates a total value.

Current Logic: It sums the unit costs and multiplies that sum by the total number of product types in the list.

Termination (Option 4): Displays a goodbye message and stops the script
